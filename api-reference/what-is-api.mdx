## API란 무엇인가요?

**API (Application Programming Interface)** 는\
서로 다른 소프트웨어가 **정해진 방식으로 데이터를 요청하고 응답할 수 있도록 하는 인터페이스**입니다.

쉽게 말해 API는 **프로그램과 프로그램이 대화하는 방법을 정의한 규칙**입니다.

예를 들어 다음과 같은 상황에서 API가 사용됩니다.

- 모바일 앱이 서버에서 사용자 정보를 가져올 때
- 웹사이트가 결제 서비스와 통신할 때
- 프론트엔드가 백엔드 서버에 데이터를 요청할 때
- 외부 서비스(Google Maps, Stripe 등)를 사용할 때

즉, API는 **서비스의 기능이나 데이터를 외부에서 사용할 수 있도록 제공하는 통로**입니다.

## API는 어떻게 동작하나요?

일반적인 API 동작 흐름은 다음과 같습니다.

1. **클라이언트(Client)** 가 서버에 요청(Request)을 보냅니다.
2. **서버(Server)** 는 요청을 처리합니다.
3. 서버가 **응답(Response)** 을 반환합니다.

Client → Request → API Server\
Client ← Response ← API Server

예를 들어 사용자의 정보를 조회하는 API는 다음과 같이 동작할 수 있습니다.

GET /users/123

서버는 다음과 같은 데이터를 반환합니다.

```json
{
  "id": 123,
  "name": "John Doe",
  "email": "john@example.com"
}
```

## API의 주요 구성 요소

### Endpoint

**Endpoint** 는 API에 접근하기 위한 URL 입니다.

예시:

https://api.example.com/users\
https://api.example.com/tasks

각 Endpoint는 특정 기능을 수행합니다.

---

### Request (요청)

클라이언트가 서버에게 보내는 데이터입니다.

요청에는 보통 다음 정보가 포함됩니다.

- HTTP Method
- URL
- Header
- Query Parameter
- Body

예시 요청:

POST /users\
Content-Type: application/json

```json
{
  "name": "Alice",
  "email": "alice@example.com"
}
```

---

### Response (응답)

서버가 클라이언트에게 반환하는 데이터입니다.

응답에는 다음 정보가 포함됩니다.

- HTTP Status Code
- Response Header
- Response Body

예시 응답:

200 OK\
Content-Type: application/json

```json
{
  "id": 124,
  "name": "Alice",
  "email": "alice@example.com"
}
```

## HTTP Method

대부분의 웹 API는 **HTTP Method** 를 사용하여 요청의 목적을 정의합니다.

| Method | 설명                          |
| ------ | ------------------------------- |
| GET    | 데이터를 조회합니다           |
| POST   | 새로운 데이터를 생성합니다 |
| PUT    | 데이터를 전체 수정합니다    |
| PATCH  | 데이터를 부분 수정합니다    |
| DELETE | 데이터를 삭제합니다          |

예시:

GET /users\
→ 모든 사용자 조회

POST /users\
→ 사용자 생성

DELETE /users/123\
→ 특정 사용자 삭제

---

## Status Code

API는 요청 처리 결과를 **HTTP Status Code** 로 반환합니다.

| Code | 의미                       |
| ---- | -------------------------- |
| 200  | 요청 성공               |
| 201  | 리소스 생성 성공       |
| 400  | 잘못된 요청            |
| 401  | 인증 필요               |
| 403  | 권한 없음              |
| 404  | 리소스를 찾을 수 없음 |
| 500  | 서버 내부 오류             |

예시:

404 Not Found

## REST API

현대 웹 서비스에서는 대부분 **REST API** 구조를 사용합니다.

REST API의 특징:

- HTTP Method 사용
- 리소스 기반 URL
- JSON 형식 데이터
- Stateless 구조

예시 REST API:

GET /users\
GET /users/id\
POST /users\
PATCH /users/id\
DELETE /users/id

## 정리

API는 **소프트웨어 간의 통신을 가능하게 하는 인터페이스**입니다.

API를 통해 다음과 같은 작업을 수행할 수 있습니다.

- 서버 데이터 조회
- 데이터 생성 및 수정
- 외부 서비스 통합
- 자동화된 시스템 구축

대부분의 현대 웹 서비스는 **REST API와 JSON 데이터 형식**을 기반으로 동작합니다.
